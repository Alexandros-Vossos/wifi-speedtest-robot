# wifi-speedtest-robot
A simple RPA bot built with UI.Vision RPA that automates an internet speed test and records the results

Description: A simple RPA bot built with UI.Vision RPA that automates an internet speed test and records the results.
It demonstrates basic web automation, conditional logic, data extraction, and writing output to an online tool.

Robot type: browser

Steps: Opens Speedtest.net and starts a speed test -> Handles cookie consent pop-up (if present) -> Extracts download and upload speed values -> Displays results in the RPA console -> Opens an online notepad and writes the results

Tool: UI.Vision RPA
Code format: Json
Robot perform: on user's demand

How to run: Install UI.Vision RPA
Copy my code and paste it in "Source View (JSON)" section
Run the macro and wait for the test to complete
View results in the console and in the notepad page opened by the bot
